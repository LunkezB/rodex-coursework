# Тестирование Rodex

## Запуск

```bash
cp .env.example .env
docker compose up --build
```

После запуска нужны эти адреса:

| Адрес                               | Что это           |
| ----------------------------------- | ----------------- |
| http://localhost:5173               | Frontend          |
| http://localhost:8000               | Backend           |
| http://localhost:8000/docs          | Swagger / OpenAPI |
| http://localhost:8000/api/v1/health | Healthcheck       |

Миграции нужно применить отдельно (после старта контейнеров):

```bash
docker compose exec backend alembic upgrade head
```

## Backend-тесты

```bash
docker compose exec backend pytest
```

Тесты лежат в `backend/app/tests/`. Каждый файл - один модуль маршрутов:

| Файл                    | Что проверяет                                 |
| ----------------------- | --------------------------------------------- |
| `test_auth.py`          | Регистрация, вход, защита эндпоинтов          |
| `test_persons.py`       | CRUD персон, изоляция данных                  |
| `test_relationships.py` | CRUD связей, циклы, ограничения ролей         |
| `test_sources.py`       | CRUD источников, привязки к персонам и связям |
| `test_sosa.py`          | Алгоритм Соса–Страдоница, предупреждения      |
| `test_reports.py`       | HTTP-эндпоинты росписи, CSV-экспорт           |
| `test_health.py`        | Healthcheck-эндпоинт                          |

## Frontend-сборка

```bash
docker compose exec frontend npm run build
```

## Тестирование вручную

Минимальная проверка основного сценария после запуска:

1. Открыть `http://localhost:5173`, зарегистрироваться с новым email и паролем.
2. Войти в приложение - должен открыться рабочий раздел.
3. В разделе "Персоны" добавить двух персон с разными именами.
4. В разделе "Связи" создать связь родитель–ребёнок между ними.
5. В разделе "Источники" создать источник и привязать его к одной из персон - убедиться, что на карточке персоны появился счётчик.
6. В разделе "Роспись" выбрать пробанда, нажать "Построить роспись" - должен появиться список поколений.
7. Переключить режим на "Дерево" - должно отрисоваться визуальное дерево предков.
8. Нажать кнопку экспорта - должен скачаться CSV-файл с результатами.
