# Rodex

Веб-приложение для ведения персон, родственных связей и архивных источников, а также для построения родословной росписи по системе Соса–Страдоница.

## Возможности

- Регистрация и вход; данные каждого пользователя изолированы от других.
- CRUD персон с поиском по всем полям (включая связанные источники), фильтрацией по полу и датам, пагинацией.
- Связи родитель–ребёнок: создание, просмотр, удаление; проверка циклов; ограничение ролей (не более одного отца и одной матери у ребёнка).
- CRUD архивных источников с привязкой к персонам и связям; счётчики источников на карточках; просмотр и управление привязками.
- Построение поколенной росписи Соса–Страдоница с просмотром в виде списка поколений и визуального дерева предков.
- Поиск и фильтрация по результатам построенной росписи.
- Экспорт росписи в формате CSV.
- Адаптивный интерфейс, skeleton loading states, PWA (manifest и service worker).

## Стек

- Backend: Python 3.12, FastAPI, SQLAlchemy 2, Alembic, PostgreSQL
- Frontend: React, TypeScript, Vite
- Infra: Docker Compose
- Tests: pytest

## Быстрый запуск

```bash
cp .env.example .env
docker compose up --build
```

После старта контейнеров применить миграции:

```bash
docker compose exec backend alembic upgrade head
```

После запуска доступны:

- Frontend: http://localhost:5173
- Backend: http://localhost:8000
- Swagger / OpenAPI: http://localhost:8000/docs
- Healthcheck: http://localhost:8000/api/v1/health

## Проверки

```bash
# Backend-тесты
docker compose exec backend pytest

# Frontend-сборка
docker compose exec frontend npm run build
```

## Основные сценарии

1. **Регистрация** - создать учётную запись по email и паролю.
2. **Вход** - войти и перейти в рабочий раздел.
3. **Персоны** - добавить персон с биографическими данными; обязательное поле - имя.
4. **Связи** - указать связи родитель–ребёнок между персонами.
5. **Источники** - добавить архивный источник, привязать к персоне или связи.
6. **Роспись** - выбрать пробанда, построить роспись, переключить между режимами список/дерево.
7. **CSV** - скачать результаты росписи в формате CSV.
